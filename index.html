Portfolio

WalkingStick

Final Project for CS50x
WalkingStick is a project developed by me as a final project for CS50x, serving as both a Chrome extension and a standalone web app. Its primary aim is to enhance internet accessibility for individuals with disabilities by offering various functionalities. Users can install the Chrome extension to redirect their new tab page to the WalkingStick web app and enable text-to-speech functionality by simply pressing a key. The web app, powered by Flask, recognizes voice commands and can provide definitions, news articles, or even play music from YouTube based on user input. This innovative solution combines simplicity and efficiency to improve internet navigation for a broader audience.
